Aplikasi **IT Asset Management System** berbasis Laravel dan Filament Admin Panel.

Project ini dibuat untuk membantu pengelolaan aset IT perusahaan secara terstruktur, mulai dari pencatatan aset, mutasi, service, retire, hingga manajemen software dan license.

---

# 🚀 Fitur Utama

## 📦 Asset Management

Mengelola seluruh aset IT perusahaan:

- Data Asset IT
- Nomor Asset IT
- Nomor Asset SAP
- Serial Number
- Vendor
- Perusahaan
- Status Asset

Fitur tambahan:

- Mutasi Asset
- Service Asset
- Retire Asset
- Software Assignment


---

## 🏢 Master Data

Pengelolaan data utama:

- Perusahaan
- Vendor
- Lokasi
- Karyawan


---

## 💻 Software Management

Mengelola software perusahaan:

- Master Software
- Software License
- Product Key
- Jumlah License
- Assignment Software ke Asset


---

## 📊 Dashboard Monitoring

Dashboard menyediakan informasi:

- Total Asset
- Asset berdasarkan status
- Asset berdasarkan perusahaan
- Warranty yang akan habis
